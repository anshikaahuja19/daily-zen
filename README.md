# The Daily Zen

## Project Description
The Daily Zen is a minimalist mental wellness web application designed to provide users with a moment of calm and focus each day. It displays an inspirational quote, a serene background image, and allows users to set a single focus task for the day.

---

##  Purpose
The goal of this project is to help users:
- Start their day with positivity  
- Maintain focus on one important task  
- Experience a calm and distraction-free interface  

---

##  APIs Used
- ZenQuotes API → for fetching daily inspirational quotes  
- Unsplash API → for fetching random calming background images  

---

##  Planned Features
- Random inspirational quote generator  
- Dynamic background image from Unsplash  
- Single focus task input  
- Refresh button for new quote and background  
- Quote category filtering (planned)  
- Search functionality for quotes (planned)  
- Breathing circle animation for relaxation (planned)  

---
